# Archives

> 历史归档区。不是默认运行路径，不代表当前 FAO 最小骨架。

---

## 目录说明

| 目录 | 内容 | 状态 |
|------|------|------|
| `memory-experiments/` | 厚记忆实验文件归档 | 历史实验，不默认加载 |
| `references/` | 本地参考资料 | PDF/大文件默认不入仓 |
| `memory-history/` | 旧工作记录归档 | 历史日期记录 |
| `whitepaper-old/` | 旧版白皮书归档 | 历史版本保留 |

---

## 规则

- 禁止从 `archives/` 自动加载技能或任务
- 轻量文本文件（README、.md、.py 源码）可入仓，保持可追溯
- PDF/大文件/二进制文件被 `.gitignore` 排除，留在本地
- 如需恢复实验，必须人工确认

---

*归档区不等于遗忘区。只是不再默认激活。*
