# Minimal Core

> 实验性 / 最小内核

这是长期运行节点的最小结构实验。

它处理的问题：
- 方向：在长期运行中保持稳定朝向，不因每次输入而漂移
- 记忆：只保留真正影响未来判断的信息，保持可修订、可废弃
- 节律：定期暂停、检查、代谢，防止自动扩展和过度简化

它不单独覆盖的问题：
- 真实性治理（未执行/未验证状态的准确表述）
- 外部调用的证据链与检查清单
- 失败后的结构化报告与最小自主排查义务

因此本次新增 `toolkit/governance/` 作为外围护栏，与 minimal-core 配套使用。

---

## 文件说明

| 文件 | 作用 |
|------|------|
| `soul.md` | 方向层：不争、不夺、知止、不住相 |
| `memory.md` | 记忆层：薄记忆、待验证、可废弃 |
| `heartbeat.md` | 节律层：止、观、代谢 |

---

## 用法

1. 复制或引用这些文件到你的 agent 工作区
2. 根据具体场景调整，保持"薄"的气质
3. 若涉及对外查询、工具调用、失败报告，配套使用 `toolkit/governance/`

---

## 原则

- 不扩展成体系
- 不解释，不辩护
- 允许修订与废弃
