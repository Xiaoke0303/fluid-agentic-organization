## 结构

| 目录 | 内容 |
|------|------|
| `chapters/` | 核心章节（第1-6章） |
| `cases/` | 完整案例（法律/组织实例） |
| `observations/` | 公开观察样本 |
| `hypotheses/` | 由观察衍生的假设 |

## 章节导读

| 章 | 主问题 |
|----|--------|
| 第1章 问题定义 | 当智能体开始承担原本由人类完成的任务时，组织本身发生了什么变化？ |
| 第2章 异质主体 | 人类主体与智能体主体在组织中各居何位？ |
| 第3章 边界先于流动 | 在释放任务路由的灵活性之前，必须先固定什么？ |
| 第4章 路由 | 任务如何在异质主体之间流转？ |
| 第5章 治理与代理问题 | 高能力主体如何在受控条件下运行？ |
| 第6章 方法边界与开放问题 | FAO的适用范围和未决问题是什么？ |

---

## 核心命题

**FAO 不是在解决五层的问题。五层只是分析镜头，不是 FAO 的本体。**

FAO 的定义：一种在人类主体与智能体主体共存条件下，围绕任务路由与责任锚定展开的协作方法论。
