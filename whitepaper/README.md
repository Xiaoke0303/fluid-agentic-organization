# FAO 白皮书

> 异质主体协作条件下的边界、路由、验证与纠错、成本约束

---

## 一句话定义

FAO（Fluid Agentic Organization，流态代理组织）是一种在人类主体与智能体主体共存条件下，围绕任务路由与责任锚定展开的协作方法论。

## 阅读说明

本白皮书当前处于第一轮成稿阶段，是理解 FAO 主线的首要入口。如需查看实现支撑，请移步仓库根目录的 `framework/` 目录。`cases/` 中的假设推演与观察样本均为支撑材料，不构成已成立结论。
