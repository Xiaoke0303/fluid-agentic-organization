# FAO 白皮书

> 异质主体协作条件下的边界、路由与治理

---

## 一句话定义

FAO（Fluid Agentic Organization，流态代理组织）是一种在人类主体与智能体主体共存条件下，围绕任务路由与责任锚定展开的协作方法论。

## 章节目录

| 章 | 主问题 |
|---|--------|
| 第1章 问题定义 | 当智能体开始承担原本由人类完成的任务时，组织本身发生了什么变化？ |

| 第5章 治理与代理问题 | 高能力主体如何在受控条件下运行？ |
