# External Research Archive

> 外部研究素材归档区。不等于 FAO 长期记忆。

---

## 内容来源

本目录存放从 memory/ 迁移的外部研究素材，包括但不限于：

- 行业人物访谈分析（黄仁勋、Jack Dorsey 等）
- 安全事件分析（Claude Code 泄露等）
- 新书/理论笔记（平克《共同知识》等）
- 项目发布观察（Anthropic Managed Agents、Hermes Agent 等）
- 其他外部研究材料

---

## 与 FAO 记忆的关系

| 关系 | 说明 |
|------|------|
| 不等于长期记忆 | 这些不是 FAO 核心判断骨架 |
| 不自动进入白皮书 | 外部素材不等于主文论证 |
| 不自动进入 memory/memory.md | 需要经过"判断-择记"流程 |
| 如需吸收 | 必须人工判断后写入对应 sidecar 或主线 |

---

## 使用方式

1. 按研究主题查找素材
2. 人工判断哪些判断值得进入长期记忆
3. 通过 `CORRECTION-WRITEBACK.md` 或 `MEMORY-INDEX.md` 正式写入
4. 不要直接引用此处文件作为 FAO 立场
