--------------------------------
一、术语校准：两个主体 → 异质主体
--------------------------------

请检查白皮书与 README 中以下表达：

- 两个主体
- 两种主体
- 双主体
- 人类主体与智能体主体（并列使用时）

处理原则：

1. “异质主体”作为更高一层的总称
   用于指代：
   - 人类主体
   - 智能体主体
   二者的共存与协作条件

2. 保留“人类主体 / 智能体主体”作为具体分类名
   即：
   - 总称：异质主体
   - 分类：人类主体、智能体主体

3. 不要机械全文替换
   只在以下位置优先修改：
   - 标题
   - 小节开头
   - 总结句
   - 定义句
   - README 的项目说明语句

4. 如果原句中“两个主体”更口语、但替换后更稳，则替换为：
   - 异质主体
   - 异质主体共存
   - 异质主体协作

5. 不要引入新术语体系
   不新增“多主体”“双元主体”等替代说法
