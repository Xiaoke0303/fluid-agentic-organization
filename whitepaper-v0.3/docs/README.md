## 核心工作句

1. **FAO 的灵魂不是层级，而是路由。**
2. **能力可以下放，责任不能消失。**
3. **不是先定义什么流动，而是先定义什么不能流动。**
4. **对外部环境产生后果的决策位置，不能流动到不能承担后果的主体。**
5. **治理不是补丁，而是存在条件。**
