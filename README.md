# FAO (Fluid Agentic Organization)

一个正在演化中的组织框架项目，关注在人类主体与智能体主体共存条件下，
如何通过任务路由与责任锚定，重构协作方式。

本仓库包含：
- [whitepaper/](whitepaper/)：完整理论说明
- [toolkit/](toolkit/)：最小运行单元与治理约束

---

## 阅读路径

| 入口 | 内容 |
|------|------|
| [whitepaper/](whitepaper/) | 完整白皮书：异质主体、边界、路由、治理 |
| [toolkit/minimal-core/](toolkit/minimal-core/) | 最小运行单元：方向、记忆、节律 |
| [toolkit/governance/](toolkit/governance/) | 真实性治理：边界暴露、外部调用检查、失败报告 |
| [toolkit/governance/templates/](toolkit/governance/templates/) | 节点接入模板：身份锚点、用户接口占位 |

---

## 一句话定位

FAO 不是抽象层级图，而是落实在具体任务中的责任安排。
