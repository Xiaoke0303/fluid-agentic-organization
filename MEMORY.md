## P0 核心共识（永不衰减）

> FAO 不是在解决五层的问题。五层只是分析镜头，不是 FAO 的本体。

**FAO 定义**：一种在人类主体与智能体主体共存条件下，围绕任务路由与责任锚定展开的协作方法论。

**核心句**：
- FAO 的灵魂不是层级，而是路由
- 能力可以下放，责任不能消失
- 不是先定义什么流动，而是先定义什么不能流动

**两种主体**：
- 人类：带宽窄、计算慢、注意力稀缺 —— 但承担最终责任、价值锚定
- 智能体：高带宽、高计算、持续处理 —— 但容易伪完成、越界、不能天然承担后果

**不能流动的**：
- 对外部环境产生后果的决策位置，不能流动到不能承担后果的主体
- 最终承担、对外代表权、边界暴露义务不能下放
- 未验证不能冒充已确认，未任不能代其责

**Soul 收口**：
> 不争、不夺、不僭、不硬。知止、不住相、守中。名实不乱、位责不僭。有核而应变，不因外物而失其位。

### 视频内容处理技能详情
- **工具链**: yt-dlp / curl (下载) → ffmpeg (音频提取/关键帧) → Whisper (转录) → LLM (总结)
- **适配平台**: 小红书(需处理风控)、B站、YouTube等
- **输出格式**: Markdown总结 + 可选PDF
- **Fallback策略**: 当下载受阻时，自动转为搜索+内容聚合模式
